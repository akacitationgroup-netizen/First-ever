Azure DevOps Project Starter

This repository is prepared as a small static site plus example pipeline files to help you get started with Azure DevOps.

Azure DevOps guide View pipeline YAML

Repos

Store code and YAML pipeline definitions here. Use the provided azure-pipelines.yml as a starting point.

Pipelines

Continuous integration using Azure Pipelines. See docs/azure-devops.md for instructions to create a pipeline and add build badges.

Boards & Artifacts

Track work items and publish build artifacts from your pipelines to support deployments.

What this repo contains

  • index.html — Azure DevOps themed landing page.
  • assets/ — styles and JS for the site.
  • azure-pipelines.yml — example pipeline to build and publish artifacts.
  • docs/azure-devops.md — steps to connect this repo to Azure DevOps and add a build badge.

Next steps

  1. Create a project in Azure DevOps and push this repository to the project's repo.
  2. Create a new pipeline and point it to azure-pipelines.yml.
  3. Add a build badge to your README using the badge snippet in docs/azure-devops.md.
